# PHP Z5Encrypt Disassembler

这个虚拟机挺复杂的，很多种反调试手段。不过手动单步运行分析，然后手动翻译是能成功的。

时间不充裕，代码是半成品，有些变量直接写在代码中了，不打算继续维护了。原本想尝试做反编译的，目前只做了反汇编。
